### Create a to-do list app.

1. Create a login and signup page using basic auth
2. Upon successful login, using the route show them their task.
3. Users can't perform CRUD operations without logging
4. Create two components.
5. One for the view which will have edit, delete,
   and add the task in the list. another component will have the input field
   (using the reactive form) to add the task name.
6. The two-component will talk through the input and output binding
